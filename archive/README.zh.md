# Archive

English version: [README.md](./README.md)

这个目录用于存放可选的历史工作区资产，这些内容不属于当前活跃的可发布面。

可以谨慎用于存放：

- 已完成 rollout 的 task packets
- 历史 verification evidence
- 为了可追踪性而保留的临时 planning 或 migration 文档

不要把 `archive/` 当作新仓库默认复制集合的一部分。

如果要做公开首版发布，除非确实需要追踪历史，否则优先只保留轻量级 archive 元数据。

对于新仓库，只复制 [README.md](../README.md) 中描述的那些受维护的 distribution surfaces。
