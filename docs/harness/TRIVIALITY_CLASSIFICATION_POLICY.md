## Content

This policy defines how to classify tasks as `trivial` or `non-trivial`.

## Quick Summary

**Default:** All tasks are `non-trivial` unless explicitly whitelisted.

**Trivial whitelist:**
- Single typo fix
- Minor comment/README clarification
- Read-only queries or status reports
- Pure formatting changes
